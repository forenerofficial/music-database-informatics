## Features\
\
- **Schema Design:** Tables for Artists, Playlists, Songs, Users, etc.\
- **Queries:** Includes joins, aggregates, and ordering queries.\
- **ERD Visualization:** Shows relationships between tables.\
- **Portfolio-ready Screenshots:** Query results and report views.\
\
## Sample Queries\
\
1. List all songs in a playlist with artist names.\
2. Count the number of songs per playlist.\
3. Top artists by appearances in playlists.\
4. Playlists created by each user.\
5. Songs ordered by date added in a playlist.\
6. Monthly listeners of artists in each playlist.\
\
## Notes\
\
- SQL files are compatible with MySQL syntax.\
- Album cover images are a small sample for demonstration.\
